# FileFlex - Sophisticated File Management App

## Installation

1. Clone the repository

```sh
git clone https://github.com/akibsaleh/FileFlex.git
cd FileFlex
```

2. Install dependencies:

```sh
npm install
```

3. Run the development server:

```sh
npm run dev
```
